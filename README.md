# dccon-downloader
디시인사이드의 콘텐츠인 디시콘 이미지를 편리하게 다운로드받을 수 있도록 하는 프로그램입니다.

# 사용법
프로그램 실행 후 디시콘 고유 번호를 입력하시면 자동으로 전체 이미지가 다운로드됩니다.

## Javascript인 경우
```
npm i
node index
```

## Python인 경우
```
pip install -r requirements.txt
python app.py
```
